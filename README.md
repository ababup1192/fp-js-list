# Preparation

[Install Node.js](https://github.com/hokaccha/nodebrew)

[Install Yarn](https://yarnpkg.com/en/docs/install)

Install packages
```
$ yarn
```

Manage packages
```
$ yarn add [package-name]
$ yarn upgrade [package-name]
$ yarn remove [package-name]
```

# Testing
```
$ yarn test
```

# Build
```
$ yarn build
// Create bundle.js to build/
```

# Build & Run
```
$ yarn buildRun
```
